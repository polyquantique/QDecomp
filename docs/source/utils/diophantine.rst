Diophantine Equation Solver
===========================

The ``qdecomp.utils.diophantine`` subpackage provides a set of tools for solving the Diophantine equation
:math:`\xi = t \cdot t^\dagger` for :math:`t \in \mathbb{D}\left[ \omega \right]` where :math:`\xi \in \mathbb{D} \left[ \sqrt{2} \right]` is given.
